# Système Argyropée pour Foundry VTT

![Foundry Version](https://img.shields.io/badge/Foundry-v13+-ff6400.svg)
![System Version](https://img.shields.io/badge/Version-1.0.5-blue.svg)

Bienvenue dans les Sentiers d'Argent. **Argyropée** est un système de jeu de rôle officiel pour Foundry VTT, conçu pour plonger les joueurs dans une cité-franche à l'aube de la révolution industrielle, où l'alchimie, les machinations de la Pègre, l'Inquisition impériale et les restes d'une magie corrompue s'entremêlent.

## ⚙️ Fonctionnalités du système

Ce système a été développé pour offrir un maximum d'automatisations et de possibilités de personnaliser les items et autres éléments.

* **Fiches de Personnages (PJ) interactives :** Gestion de la Santé, du Panache, de l'Initiative, de l'Équipement (encombrement), de la Magie et de la Richesse.
* **Fiches de PNJs & Bestiaire :** Fiches allégées pour les PNJs, et fiches spécifiques pour les créatures.
* **Gestion complète de l'inventaire :** Armes, Armures (avec calcul de protection et d'usure), Consommables, Pièges (avec état de déploiement) et Sortilèges.
* **Cartes de Chat personnalisées :** Chaque jet de compétence, d'attaque, de sort ou de dégâts génère une carte de chat personnalisée et colorée selon le thème de l'action (Magie, Combat, Alchimie, etc.), avec calcul automatique des réussites critiques, échecs et échecs critiques.
* **Éditeur de Journal enrichi (ProseMirror) :** Un menu "Argyropée" exclusif a été intégré nativement dans l'éditeur de texte de Foundry. Il permet de formater vos journaux avec les styles intégré dans le système (et que vous pouvez donc modifier directement dans le CSS si vous le souhaitez).

Le système est livré prêt à jouer avec des compendiums pour tout ce qui est utile (Archétypes de PNJs, Créatures, Objets en tout genre, métiers...).
